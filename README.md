# Filco_FKB-109J.BIG
Retrofit PCB for Strong Man SMK-8851 and related full-sized keyboards.

## Planned support:
- AEK (US, EUR, JPN)
- DELL AT101W (US, EUR, JPN)
- DELL AT101 (_READ BELOW._)
- Strong Man SMK-8851 (BAE, JPN)
